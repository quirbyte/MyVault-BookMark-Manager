**MyVault** is a minimalist, high-performance bookmark manager designed for developers and power users. It features a sleek "glow-card" interface, real-time search, and a favoriting system to keep your most important links organized and accessible.

## âœ¨ Features

- **Glow-Card UI**: Modern card design using custom Tailwind shadows for depth and clarity.
- **Real-time Search**: Instant filtering of bookmarks by title or category via backend query parameters.
- **Favorites System**: Toggle "Favorite" status using optimized `HTTP PATCH` requests.
- **Smart Redirects**: Automatic URL handling to ensure links open safely in new tabs.
- **Responsive Grid**: Fluid layout that adapts perfectly from mobile to desktop.
